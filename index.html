Svelte

¿Qué es Svelte?

( Un framework de JS que nos facilita la escritura de código para el desarrollo web. )

Svelte es un marco de trabajo de código abierto para construir interfaces de usuario interactivas en la web. A diferencia de otros marcos como React o Vue, Svelte realiza la mayor parte del trabajo durante la compilación en lugar de en tiempo de ejecución. Esto significa que en lugar de enviar una biblioteca voluminosa al navegador, Svelte compila tu aplicación en código JavaScript optimizado y altamente eficiente.

Con Svelte, escribes código usando un lenguaje similar a HTML con características adicionales para la lógica y el manejo de eventos. Luego, este código se compila en código JavaScript altamente optimizado que actualiza el DOM de manera eficiente. Esta aproximación le da a Svelte un rendimiento muy rápido y un tamaño de paquete pequeño, lo que lo hace ideal para aplicaciones web de alto rendimiento.

¿Por qué Svelte?

Hay varias razones por las que alguien podría elegir Svelte sobre otros marcos de trabajo de JavaScript para el desarrollo de aplicaciones web:

Rendimiento
Svelte realiza la mayor parte del trabajo durante la compilación en lugar de en tiempo de ejecución, lo que resulta en un código más optimizado y un rendimiento más rápido en comparación con otros marcos que realizan más trabajo en el navegador.
Tamaño del paquete
Debido a su enfoque de compilación, las aplicaciones Svelte tienden a tener un tamaño de paquete más pequeño en comparación con otros marcos, lo que las hace más rápidas de cargar y más eficientes en el uso de los recursos del cliente.
Sintaxis intuitiva
Svelte utiliza una sintaxis similar a HTML, lo que facilita la escritura y comprensión del código para muchos desarrolladores, especialmente aquellos que ya están familiarizados con HTML y CSS.
Menos abstracción
Svelte es menos abstracto que otros marcos, lo que significa que puedes trabajar directamente con el DOM y tener un mayor control sobre cómo se actualiza y renderiza tu aplicación.
Facilidad de aprendizaje
Debido a su sintaxis intuitiva y enfoque menos abstracto, Svelte puede ser más fácil de aprender para aquellos nuevos en el desarrollo web o para aquellos que desean una curva de aprendizaje más suave.

En resumen, Svelte ofrece un enfoque diferente para el desarrollo de aplicaciones web que prioriza el rendimiento, el tamaño del paquete y la facilidad de uso, lo que lo hace una opción atractiva para muchos desarrolladores.

¿Cómo instalar NodeJS?

Para instalar NodeJS descargamos en isntalador desde la página oficial y lo ejecutamos. A continuación, seguimos los pasos del instalador para completar la instalación de NodeJS en nuestro sistema.

¿Cómo crear un proyecto en Svelte?

Una vez instalado NodeJS, podemos crear un nuevo proyecto de Svelte utilizando la herramienta de línea de comandos npm que viene con NodeJS. Para crear un nuevo proyecto de Svelte, ejecutamos el siguiente comando en la terminal: 

                
    npm create svelte@latest NOMBRE_DEL_PROYECTO

Cambiamos a la carpeta del proyecto recién creado: 

                
    cd NOMBRE_DEL_PROYECTO

Y luego instalamos las dependencias del proyecto: 

                
    npm install

Con esto, hemos creado un nuevo proyecto de Svelte y estamos listos para comenzar a desarrollar nuestra aplicación web.

Si queremos ver nuestra aplicación en funcionamiento, podemos ejecutar el siguiente comando en la terminal: 

                
    npm run dev

Entender Svelte por partes

En Svelte hay tres partes principales que debemos conocer para poder desarrollar aplicaciones web con él:

Script
Donde escribimos la lógica de nuestra aplicación en JavaScript/TypeScript.
HTML
Donde escribimos la estructura de nuestra aplicación en HTML.
Style
Donde escribimos los estilos de nuestra aplicación en CSS.
                
    <script>
        // Lógica de la aplicación
    </script>
    
    <!-- Estructura HTML -->

    <style>
        /* Estilos de la aplicación */
    </style>

Crear nuestro primer componente

Quiero hacer un botón que tenga como mensaje "Hola Svelte!" y cuando se le haga click cambie el mensaje a "Hola Mundo!". 

                
    <script>
        let word = 'Hola Svelte';
    
        function handleClick(){
            word = 'Hola Mundo!'
        }
    </script>
    
    <button on:click={handleClick}>{ word }</button>
    
    <style>
        button{
            background: red;
            color: black;
        }
    </style>

Declaraciones reactivas

Creamos el tipico botón que al hacer click aumenta un contador. 

                
    <script>
        let counter = 0;
    
        function handleClick(){
            counter++
        }
    </script>
    
    <button on:click={handleClick}>{ counter }</button>
    
    <style>
        button{
            color: black;
        }
    </style>

Pero ahora le queremos añadir que nos diga si es par o impar el número que muestra 

                
    <script>
        let counter = 0;
    
        function handleClick(){
            counter++
        }
    
        let isEvenOrOdd = counter % 2 === 0 ? 'Par' : 'Impar'
    </script>
    
    <button on:click={handleClick}>{ counter }{isEvenOrOdd}</button>
    
    <style>
        button{
            color: black;
        }
    </style>

El error que tenemos aqui es que la variable isEvenOrOdd no se actualiza cuando el contador cambia, para solucionar esto debemos hacer que sea una variable reactiva.

Para poder hacer que sea una variable reactiva usamos $: en vez de let. 

                
    <script>
        let counter = 0;
    
        function handleClick(){
            counter++
        }
    
        $: isEvenOrOdd = counter % 2 === 0 ? 'Par' : 'Impar'
    </script>
    
    <button on:click={handleClick}>{ counter } {isEvenOrOdd}</button>
    
    <style>
        button{
            color: black;
        }
    </style>

Pero esta declaración reactiva es muy simple, si queremos ejecutar un bloque de código podemos usar $: {}. 

                
    <script>
        let counter = 0;
    
        function handleClick(){
            counter++
        }
    
        $: isEvenOrOdd = counter % 2 === 0 ? 'Par' : 'Impar'
        $: {
            if (counter > 10){
                counter = 10
            }
        }
    </script>
    
    <button on:click={handleClick}>{ counter } {isEvenOrOdd}</button>
    
    <style>
        button{
            color: black;
        }
    </style>

Fetching

Queremos capturar el input de un usuario y mostrarlo en la consola. 

                
    <script>
        let value = '';
        function handleInput(event){
            value = event.target.value
        }
        // Aqui con una declaración reactiva mostramos el valor en consola
        $: console.log(value)
    </script>
    
    <input value={value} on:input={handleInput} placeholder="Search ID..."/>
    
    <style>
    </style>

Sabiendo como capturar el input de un usuario, ahora podemos usarlo para hacer peticiones a nuestra API.

En este caso, usaremos una API sencilla que es para productos.

https://products-api-fastapi.onrender.com/ 
                
    <script>
        let value = '';
        function handleInput(event){
                value = event.target.value
        }
    
        $: if (value !== ''){
            // Aqui usamos el endpoint que la api nos da para traer solo 1 producto a base del ID que se ingrese
            fetch(`https://products-api-fastapi.onrender.com/products/${value}`)
            .then(res => res.json())
            .then(apiResponse => {
                console.log(apiResponse)
            })
        }
    </script>
    
    <input value={value} on:input={handleInput}/>

Renderizado condicional

El renderizado condicional es una técnica que nos permite mostrar u ocultar elementos en función de una condición. En Svelte, podemos lograr esto utilizando las directivas:

  • {#if} que va acompañada de {/if} para indicar el fin del if
  • {:else} lo ponemos para manejar este tipos de casos.
                
    {#if condition}
        <span>Mensaje</span>
    {:else}
        <span>Mensaje</span>
    {/if}
Renderizar lista de elementos

Podemos usar la directiva {#each} para recorrer una lista de elementos y renderizarlos en nuestra aplicación. La directiva {/each} se utiliza para indicar el final del bucle. 

                
    <script>
        let miLista = [
            {
                "title": "Laptop",
                "precio": 1000,
                "imgURL": "https://http2.mlstatic.com/D_NQ_NP_850508-MLU72263308719_102023-O.webp"
            },
            {
            "title": "Mouse",
            "precio": 500
            },
            {
            "title": "Monitor",
            "precio": 700
            },
            {
            "title": "Teclado",
            "precio": 499
            },
            {
            "title": "Audifonos",
            "precio": 300
            }
        ]
    </script>
    
    {#each miLista as item}
        <div>
            {#if item.imgURL}
		<img src={item.imgURL} alt={item.title}/>
            {/if}
            <p>{item.title}</p>
            <p>${item.precio}</p>
            <hr/>
        </div>
    {/each}
    
    <style>
    </style>
